RentNaija Site Plan

Your trusted platform for renting homes and booking stays in Nigeria

Site Name

RentNaija

This name clearly represents a rental platform focused on Nigeria. It is short, memorable, and directly communicates the purpose of the platform to users.

Site Purpose

The website will serve as a platform where users can search for rental properties, book short-term stays, and connect directly with landlords or agents. It aims to simplify the renting process in Nigeria by providing verified listings, messaging features, and easy application processes.

Scenarios

  • How can I find affordable apartments in my preferred city quickly?
  • How do I contact a landlord or agent without being scammed?
  • Can I book a short-term stay like a hotel through this platform?

Color Schema

  • #0A2540 – Used for headers, navigation bar, and footer (Primary color)
  • #2EC4B6 – Used for accents, buttons, and highlights
  • #F4F4F4 – Used as background color

Typography

  • Segoe UI – Used for headings
  • Arial – Used for body text

Wireframes

Mobile View

Top Section: Logo + Menu Icon

Search Bar: Location + Price Filter

Main Content: Property Cards (image, price, location)

Call to Action: View Details Button

Footer: Contact + Links

Desktop View

Header: Logo + Navigation Menu

Hero Section: Large Search Bar

Layout: Sidebar Filters (left) | Property Listings Grid (right)

Extras: Featured Listings Section

Footer: Links, Contact Info, Social Media

Testing

This document will be tested using W3C HTML and CSS validators to ensure no errors. Additional checks will include accessibility, SEO best practices, and performance optimization.
